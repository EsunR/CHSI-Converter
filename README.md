# 学信网档案英文版转化器

用于将[学信档案](https://my.chsi.com.cn/archive/bab/index.action)一键转化为可用于[GitHub 学生认证](https://education.github.com/discount_requests/application)的英文版文件

## 认证步骤：

### 1. 个人档案

- 学信网下载学籍验证报告
- 使用本项目或有道等提供文档翻译的软件，将 pdf 报告转化为 word 文档
- 打开翻译后的文档，调整至大小合适
